# Kairos Public Sources

> Status: v0.1 seed layer
> Last reviewed: 2026-07-08

This folder contains curated public evidence links for Kairos Atlas.

It is not the internal evidence workspace. It does not expose raw internal evidence packs, model-search transcripts, private notes, or unreviewed source-repair material.

## Purpose

The public source layer helps readers inspect why a Kairos claim is supported, contested, watchlisted, or still under source repair.

Each public source should make clear:

- what claim it supports;
- what it cannot prove;
- whether it is official data, central-bank material, academic work, company filing, market data, or secondary material;
- whether it is ready to show publicly or still needs source repair.

## Relationship to Kairos Files

```text
Internal Evidence Packs
-> Public Curated Source Registry
-> Website Source Inspector
-> Public Research Briefs
```

The registry supports the public website and research briefs. It does not replace the private research workspace.

## Guardrails

- AI is not a confirmed K6.
- K6 remains Candidate Watch.
- Watchlist items are not confirmed facts.
- Source-repair items cannot be displayed as supported claims.
- External model output is not a source.
- Google search wrapper links, weak encyclopedic links, and unverified market-blog links should not be used as core evidence.
