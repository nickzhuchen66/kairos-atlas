# Kairos · 周期天象图

[English](README.md) | **简体中文**

**Kairos** 是一个公开的宏观周期观测网站，用于展示当前宏观 regime、长波背景、观察清单、资产含义和证据门控复盘。

中文名：**周期天象图**  
英文副标题：**Macro Cycle Observatory**

公开仓库名：**kairos-atlas**  
正式网站：**[kairos-atlas.com](https://kairos-atlas.com/)**

`kairos-atlas` 用作公开仓库和域名身份；`Macro Cycle Observatory` 继续作为产品英文副标题，因为当前网站更接近一个“宏观周期观测仪表盘”，而 Atlas 更适合作为公开入口和 Long Wave Atlas 的命名锚点。

Kairos 的目标是把宏观周期判断做成可点击、可复盘、可证伪的观测仪，而不是把复杂判断包装成单一预测。

## 为什么做 Kairos

长波周期和宏观周期的资料很丰富，但真正有用的材料散落在书籍、论文、市场框架、金融史、政策史和当前数据里。很多康波阶段的表层事件不同，但底层结构经常相似：

- 信用扩张与再融资压力；
- 技术扩散与资本开支浪潮；
- 货币制度变化与财政冲击；
- 通胀波动、美元流动性与资产重估；
- 投机融资、金融脆弱性，以及后续的验证或证伪。

Kairos 试图把这些反复出现的结构整理成一张公开的宏观周期天气图。它不是日期预言机，也不是交易信号系统，而是一个帮助学习历史周期、观察当前 regime、比较历史与当下的研究展示层。

## 理论来源

Kairos 背后有一组私有理论阅读和提取笔记，用于综合长波理论、债务周期、金融不稳定理论、货币史和资产 regime 框架。

当前公开版本受 Schumpeter、Minsky、Kindleberger、Dalio、Howard Marks、Bordo / Eichengreen、Hélène Rey、Merrill Lynch Investment Clock，以及中文长波周期研究传统启发。

私有理论提取笔记不会进入公开仓库。它们作为研究底座存在；公开仓库只展示整理后的结构、当前观察项、证据状态和可证伪的判断边界。

## Research Lineage

Kairos 将理论底座压缩成几组公开可读的分析镜头：

| 谱系 | 代表来源 / 传统 | 对 Kairos 的作用 |
|---|---|---|
| 长波与创新周期 | Schumpeter；中文长波周期研究，包括周金涛 | 支撑 K4 / K5 / K6 candidate 的阶段表达，以及技术扩散是否成立的问题。 |
| 债务周期与危机机制 | Ray Dalio；Minsky；Kindleberger / Aliber / McCauley | 区分 hedge finance、speculative finance、脆弱性、狂热、恐慌和重置。 |
| 市场周期与资产行为 | Howard Marks；Merrill Lynch Investment Clock；资产 regime 研究 | 把宏观 regime 连接到资产表现，但不把网站变成交易建议。 |
| 货币制度与美元流动性 | Bordo / Eichengreen；Eichengreen；船桥洋一；Hélène Rey | 连接布雷顿森林、黄金约束、全球美元流动性和跨境金融周期。 |
| 经济史与商业周期综合 | DeLong；Tvede；Ellis / Goldman-style forecasting frameworks | 让历史比较落在可复用结构上，而不是只做叙事类比。 |
| 黄金与货币信任 | Erb / Harvey；货币史研究 | 把黄金视为随 regime 变化的信号，而不是单一真实利率工具。 |

这份 lineage 是影响地图，不是公开笔记库。公开仓库保留综合框架和判断脚手架；私有提取笔记继续留在发布包之外。

## Watchlist 精华

- **AI as K6 Candidate**：AI 可能成为新长波扩散引擎，但在跨行业生产率证据成立前，必须保留为 `Candidate Watch`。
- **AI capex 融资脆弱性**：hyperscaler 的资本开支仍可能由强现金流支撑，但独立 GPU cloud 和数据中心融资需要单独观察脆弱性。
- **高真实利率**：2020 后的利率 regime 可能挑战低利率资产定价体系，但这仍是未完成判断，不是最终结论。
- **高利率下的宽松金融条件**：高政策利率不自动等于金融条件收紧，信用利差、流动性和风险偏好仍然重要。
- **黄金与法币信用对冲**：黄金可能从单纯真实利率资产转向更广义的法币信用对冲资产，但公开证据底座仍需继续增强。

查看 [`research/`](research/) 获取公开研究摘要。

## Kairos 展示什么

- **Current Regime**：当前宏观周期天气图。
- **Cycle Stack**：宏观时钟、金融周期、流动性 / 政策、全球美元、长波背景、市场验证六层结构。
- **Long Wave Atlas**：K4 / K5 历史阶段和 K6 Candidate Watch。
- **Watchlist**：可能改变当前判断的观察项。
- **Asset Behavior Map**：不同资产在当前 regime 下的含义。
- **Evidence Ledger**：已支持、有争议、未确认、待补证和复盘记录。

## Kairos 不是什么

Kairos 不是：

- 日期预言机；
- 交易信号系统；
- 投资建议；
- 对 K6 已确认的声明；
- 自动根据新闻流更新宏观判断的网站。

## 产品护栏

- **K6 仍然是 Candidate Watch**，不是 confirmed K6。
- Watchlist 是观察项，不是 confirmed fact。
- 有争议判断必须保留争议状态。
- 部分核验 / 来源补核中的内容不能展示成 fully supported。
- 宏观读数应保留证据状态、置信度、观察项和证伪条件。
